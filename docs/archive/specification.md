## 1. Scope
This directory contains the foundational visual elements of the **Foliotype Protocol**. These assets are the "Masters" and must remain unaltered to ensure brand consistency across all platforms.

## 2. Core Assets
* **`votre_logo.svg`**: The primary monumental logotype. Vector-based for infinite scalability.
* **`banniere_haute_def.mp4`**: The high-definition master of the "Breath" animation (1.05 MB).
* **`foliotype-mark.svg`**: The simplified iconic mark for UI/UX applications.

## 3. Technical Standards
* **Format:** Vector (SVG) for logos / H.264 (MP4) for motion.
* **Color Space:** Strictly aligned with the Foliotype digital palette.
* **Storage:** Standard Git tracking for assets < 2MB to ensure native browser rendering within GitHub.
